# 诺奖物理互动展厅项目规划与要求

更新时间：2026-06-22

## 1. 作业目标

- 第二项作业形式不限，评价重点为：内容准确性、课程关联性、作品创新性。
- 作品不写成论文，定位为可打开、可互动、可展示的高级网页展厅。
- 主题暂定：诺奖物理通识互动展厅。
- 截止日期：2026-07-10 18:00。

## 2. 总体定位

- 不是学生心得页，也不是论文可视化。
- 是一个“刚走进高级科学博物馆”的数字互动展厅。
- 气质关键词：黑盒美术馆、科学博物馆、精密实验装置、低调技术力、公共展览。
- 首页要有高级动态背景和互动反馈，但不能像科幻游戏大厅。
- 首页不放哲学名言。
- 哲学元素以各展区的“墙面铭文”形式出现，且必须是某位哲学家真实说过的话，正式使用前逐条核来源。

## 3. 视觉原则

- 主方向：黑盒展览 / 高级科学博物馆。
- 避免：廉价赛博风、发光球、HUD 堆料、紫蓝 AI 渐变、卡片堆砌、学生 PPT 风、游戏登录页风。
- 首页空间应像进入一个高级科学馆：暗色大空间、克制冷光、微粒空气感、材质和尺度感。
- 入口页以最初“高级博物馆入口”的方向为准：短标题、少量英文标识、暖白偏金的展馆光影、小幅空间纵深。避免线条割裂空间，也避免低矮小房间感。
- 不使用显眼横纵轴、地面网格、框内外色块差异或大面积 HUD 式图形；光效必须像环境光和空气里的投影，不像贴图边框。
- 入口页只保留一句很短的欢迎词和极轻的进入控件，字体不能占据屏幕主体。
- 所有页面共享同一设计语言。
- 物理学家照片统一处理：黑白或低饱和银盐质感、深色背景、统一裁切、轻微颗粒、博物馆展柜光。
- 当前首页入口和导览页已进入“低噪声但有纵深空间”的美术方向；后续功能扩展不能破坏少文字、少 HUD、强空间层次的入口气质。
- 首页到各展厅、各展厅之间切换，必须有“进入展厅”的过渡动画，不能只是普通链接跳转。
- 过渡动画方向：旧页文字与界面先柔和退场，新页内容再从空气光层中浮现；节奏应有约半秒的“漫步进入”感，不能闪切，也不能拖沓或突兀白光。光圈扩散只适合手势模式开启，不作为主要场景切换。
- 过渡动画实现禁忌：不能用全局规则强行覆盖 `main` 直接子元素的 `transform`、`opacity` 或 `filter`。许多展厅主体、光层、颗粒层依赖这些属性定位和成像；场景进入应优先使用独立遮罩层淡出，离开时只做短暂雾化消散。
- 场景切换不能出现黑屏或近似黑屏；博物馆漫步感应当像空气光层短暂掠过，而不是电影式 blackout。避免全屏大 blur 和高不透明遮罩，旧页应保留可见连续性，新页轻柔接上。
- 性能原则：入口、导览、普通环境粒子不应高成本绘制。Canvas 背景默认 DPR=1，少粒子、少大面积渐变；只有用户正在交互的核心展品才允许更高频率的视觉更新。尺度透镜是已封版的核心连续缩放展品，必须保留 `devicePixelRatio` 高清绘制，不参与背景页降级优化。

## 3.1 哲学铭文规则

- 首页不放哲学名言。
- 具体展厅可以放哲学铭文，但要小、克制，像墙面铭牌或投影脚注。
- 哲学铭文可以做成多句相关哲学名言循环显示。
- 必须写出出自哪位哲学家或思想家；不需要写具体网络出处。
- 所有名言必须是某位哲学家真实说过的话；正式提交前逐条核查原文、译文和出处。
- 铭文服务于展厅主题，不抢主互动的视觉重心。

## 4. 内容架构初稿

### 4.1 首页与导览：观测之门

- 当前首页拆成两层：
  - `prototype/index.html`：极简入口页，只承担“刚走入高级科学博物馆”的第一印象，不堆展厅入口。
  - `prototype/hall.html`：馆内导览页，承担展厅导航，视觉像安静的博物馆导视，不做传统卡片堆砌。
- 入口页表现：暖白偏金的博物馆光、静音、空调清凉感、克制未来感、自动流动光影、前后纵深感；左上/左下可保留少量英文环境标识；不写详细介绍。
- 导览页表现：回到馆内大厅导视方向，不做书本目录；背景不能出现上下割裂的色块分界；展厅入口可有弱边界或默认无边界，鼠标悬停时再显出边框和微光；避免过密排版。
- 导览页背景层必须是连续全屏光场，不能用从半屏/下半屏开始的墙面或地面色块制造空间感；hover 效果应是局部微光，不应出现矩形色块跨过背景断层。
- 首页和导览页都不放哲学名言；铭文只在具体展厅出现。
- 首页和导览页都不使用手势，避免入口体验变成炫技界面。

7. 第七展厅：时间与共振。

### 4.2 普通展厅

控制在 5 个以内，避免内容失控。

1. 光之展厅：看见不可见
   - 对应：光量子、显微、激光、结构色、成像。
   - 互动：显微调参台，调节波长、焦距、噪声、荧光标记、入射角。

2. 量子展厅：测量之前
   - 对应：黑体辐射、老量子论、波粒二象性、测量、概率解释。
   - 互动：双缝/坍缩剧场，打开或关闭探测器，概率云变为确定结果。

3. 身体成像展厅：被重构的身体
   - 对应：X 光、核磁共振、医学成像。
   - 互动：人体切片，切换 X 光/MRI/PET/NMR 等视图。

4. 时间与共振展厅：世界的节拍器
   - 对应：共振、原子钟、频率、相对论校准。
   - 互动：调频仪，频率接近固有频率时响应增强；原子钟校准漂移。

5. 算力与微缩展厅：从量子比特到芯片雕刻
   - 对应：量子计算、晶体管、集成电路、摩尔定律、蓝光 LED。
   - 互动：经典/量子算力对比；晶体管结构从平面到 FinFET/GAA 的演化。

### 4.3 特殊展厅一：物理学家走廊

- 不是哲学家走廊。
- 用户像走在一条高级暗色肖像长廊。
- 左右陈列课程中提到的重要物理学家照片、成果、轶事、课程关联。
- 靠近或悬停时，照片、成果、关键实验、性格片段、相关展厅入口逐层浮现。
- 当前已调整为第二展厅，并集成人格测试图鉴系统。
- 图鉴收集机制：
  - 人物档案默认全部开放，不需要测试解锁。
  - 人格测试完成后，将匹配到的物理学家 id 写入 localStorage，用于解锁对应专属信物。

  - 每位物理学家的“信物”只在测试触发对应结果后点亮。
  - 走廊需要提示“怎样更容易测到这个人物”，但不能把某道题设计成专门服务某个人。
  - 信物可以用简单小图腾表达，每个人不同，保持美观、统一、博物馆式。
  - 记忆功能必须持久化，关闭网站后不能清零；当前使用 localStorage 保存。

## 5. 手势控制策略

- 可以实现电脑摄像头前的手势控制。
- 推荐使用 MediaPipe Web 方案识别手部关键点或手势。
- 手势控制作为增强体验，而不是唯一操作方式；必须保留鼠标/键盘/触控兜底。
- 首页和物理学家走廊暂不使用手势操作，避免把入口和人物叙事做成炫技界面。
- 手势主要放在 2-3 个普通展厅中。
- 这些展厅默认使用非手势操作，通过一个高级且明确的按钮进入“手势操作状态”。
- 进入手势状态时需要有一个短而震撼的仪器启动/空间校准动效。
- 手势状态必须能通过手势退出，也必须保留鼠标按钮退出。
- 尺度展厅手势语义：
  - 张开单手：向微观尺度缩放，更能看见细节。
  - 五指合拢：向宏观/宇宙尺度缩放。
  - 单手握拳：暂停手势操作，便于停下观察；改变为张开或合拢手势后继续缩放。
  - 单手握拳保持约 2 秒：可靠退出手势模式。
  - 不使用响指退出；普通摄像头对响指瞬间不稳定，容易误判或漏判。
- 浏览器摄像头权限必须由用户主动开启。
- 手势模块可做成可选开关：开启后页面显示低调的校准提示。
- 性能原则：只在首页、物理学家走廊、1-2 个核心展品使用手势；其他展区使用普通交互。
- 性能原则更新：只在 2-3 个普通展厅使用手势；首页和物理学家走廊不用手势。
- 第一优先手势试点：尺度展厅。双手拉开/合拢用于缩放尺度，鼠标滚轮/滑杆作为默认操作。
- 第一优先手势试点已调整：尺度展厅使用单手张开/合拢控制缩放，握拳暂停，双手击掌两次退出。
